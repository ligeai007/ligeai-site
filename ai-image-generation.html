一、即梦AI配置与使用

1.1 即梦AI简介

即梦是字节跳动旗下的AI图片生成平台，基于火山引擎提供高质量图片生成服务。支持文生图、图生图、字体设计等多种模式。

官网：https://jimeng.jianying.com

1.2 API获取

1. 登录 火山引擎控制台
2. 实名认证（个人开发者有免费额度）
3. 开通"图片生成"服务 → 创建Access Key和Secret Key

1.3 环境配置

# Windows PowerShell
$env:JIMENG_ACCESS_KEY="你的AccessKey"
$env:JIMENG_SECRET_KEY="你的SecretKey"

1.4 文生图命令

node openclaw-skills/ken-jimeng-image-gen/scripts/jimeng.mjs generate \
  --prompt "一只可爱的卡通熊猫，穿着宇航服，在月球上玩耍，动漫风格" \
  --ratio 16:9 \
  --save ./output.png

1.5 高质量出图公式

[主体描述] + [场景/环境] + [风格] + [光线] + [视角] + [渲染质量]

示例：一只穿着汉服的少女站在故宫红墙前，飘落的花瓣，傍晚金色阳光，电影感色调，写实摄影风格，广角视角，8K超清

1.6 4K高清模式

--ratio 16:9-4k --force-single

1.7 字体设计

node jimeng.mjs generate \
  --text "新年快乐" \
  --color 红色 \
  --illustration "烟花,灯笼,梅花" \
  --ratio 4:3 \
  --save ./font_design.png

二、火山引擎图片生成API

2.1 HTTP请求示例

curl -X POST https://image.volcengineapi.com/api/v1/text2image \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "model": "image-01",
    "prompt": "一只可爱的橘猫在阳光下打盹，写实摄影风格",
    "aspect_ratio": "16:9",
    "return_url": true
  }'
💡 提示：火山引擎图片生成API支持多种比例，包括1:1、4:3、16:9、9:16等。

三、ComfyUI工作流

3.1 安装

git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI
pip install -r requirements.txt
python main.py

访问 http://127.0.0.1:8188 打开界面

3.2 基础工作流节点

文生图需要连接：CLIP Text Encode（正面）→ Empty Latent Image → KSampler → VAE Decode → Save Image

3.3 推荐参数

参数推荐值说明
Steps25-35采样步数
CFG7-9提示词引导强度
Samplerdpmpp_2m采样器
Denoise1.0文生图=1.0

四、商业应用实战

4.1 小红书配图

[产品/主体] + [场景] + [风格：简约/清新/高级感] + [配色：低饱和] + [比例：3:4竖版]

示例：一杯手冲咖啡放在木桌上，旁边有打开的笔记本和眼镜，早晨阳光，小清新风格，马卡龙色调，3:4竖版

4.2 抖音封面图

要点：竖屏9:16、大字标题、高对比度、人物+场景结合

4.3 旅行海报

九寨沟五彩池风景，碧绿的海子倒映着雪山，秋天层林尽染，电影感色调，中国山水画意境，16:9横版

4.4 产品主图（电商）

极简风格保温杯，白色背景，正面45度角展示，光滑金属质感，商业摄影风格，纯白底，8K超清

五、版权与合规

5.1 商用注意事项

  • 即梦、火山引擎生成的图片可商用（具体看平台协议）
  • ComfyUI使用SD系列模型，商用需注意模型许可
  • 涉及人脸的图片需确保有权使用
  • 避免生成名人、商标等受保护内容

5.2 内容审核

以下内容会被拦截：真实人脸（尤其名人）、暴力血腥、政治敏感、低俗色情。
