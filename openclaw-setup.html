一、安装OpenClaw

1.1 环境准备

安装前请确认已安装Node.js 18+（推荐Node 20）：

node -v    # 应显示 v18.x 或更高
npm -v     # 应显示 9.x 或更高

下载地址：https://nodejs.org

1.2 安装OpenClaw

npm install -g openclaw
openclaw --version

1.3 初始化配置

openclaw init

按提示填写名字、时区（Asia/Shanghai）、默认模型（建议MiniMax-M2.7）。

二、配置文件详解

2.1 配置文件位置

Windows: C:\Users\用户名\.openclaw\openclaw.json

2.2 核心配置项

{
  "gateway": {
    "port": 18789,
    "mode": "local",
    "bind": "loopback"
  },
  "models": {
    "providers": {
      "minimax": {
        "apiKey": "你的API Key"
      }
    }
  }
}

2.3 模型配置示例

"minimax": {
  "baseUrl": "https://api.minimaxi.com/anthropic",
  "apiKey": "sk-你的密钥",
  "models": [{
    "id": "MiniMax-M2.7",
    "contextWindow": 200000
  }]
}
💡 提示：MiniMax API Key在官网申请，个人开发者有免费额度。

三、技能（Skills）管理

3.1 技能安装

技能存放目录：~/.openclaw/skills/

每个技能是一个独立文件夹，包含 SKILL.md 说明文件。

3.2 常用技能推荐

技能名称功能
travel-planner-pro旅行规划、景点推荐
douyin-publisher自动发布视频到抖音
xiaohongshu-post图文笔记自动发布
feishu-doc读写飞书云文档
image-genAI绘图工具

四、浏览器自动化配置

4.1 启用浏览器

"browser": {
  "enabled": true,
  "defaultProfile": "user"
}

4.2 Chrome远程调试

# Windows - 关闭Chrome后执行
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

启动Chrome后保持运行，再启动OpenClaw即可自动连接。

4.3 验证连接

openclaw browser status

显示 "Chrome: ok" 即表示连接成功。

五、飞书集成配置

5.1 创建飞书应用

1. 进入 open.feishu.cn/app
2. 创建企业自建应用
3. 获取 App ID 和 App Secret

5.2 配置示例

"channels": {
  "feishu": {
    "enabled": true,
    "appId": "cli_xxxxxxxx",
    "appSecret": "你的AppSecret"
  }
}

七、常见问题

Q1：openclaw命令找不到？

解决：检查npm全局bin目录是否加入PATH

npm config get prefix
# 将输出的路径加入系统PATH

Q2：浏览器连接失败？

解决：确保Chrome已开启远程调试模式，版本≥144，端口9222未被占用。

Q3：飞书消息无法发送？

解决：检查App ID和App Secret是否正确，确认应用已启用机器人能力。
