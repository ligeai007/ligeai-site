一、内容策划：爆款选题公式

1.1 选题三问

  • 用户为什么要看？ — 提供了什么价值（实用/情绪/审美）
  • 为什么现在看？ — 时效性、热点关联
  • 看完为什么要关注？ — 差异化、记忆点

1.2 爆款选题模型

类型模板例子
实用价值型[技能/方法论] + [效果/收益]"3个ChatGPT提示词，让文案效率提升10倍"
情绪共鸣型[痛点场景] + [情感宣泄]"30岁转行互联网，被说太晚..."
热点借势型[热点事件] + [自己角度]"淄博烧烤爆火，总结5个蹭热点方向"

1.3 各平台选题差异

平台内容倾向爆款类型
小红书年轻女性为主干货教程、好物种草、生活方式
抖音全年龄段剧情搞笑、技能展示、热点资讯
微信视频号中老年为主温情故事、正能量、新闻资讯

二、AI辅助脚本创作

2.1 爆款标题公式

  • 数字型："5个方法教你..."、"3分钟搞定..."、"99%的人不知道..."
  • 对比型："别人vs你的差别"；"之前vs之后"
  • 悬念型："原来这才是..."、"没想到这么容易..."

三、剪映（CapCut）剪辑教程

3.1 基础操作

  • 分割视频：移动播放头 → 按S键或点击"分割" → 删除不需要片段
  • 添加转场：点击片段连接处 → 选择转场效果（推荐闪黑、溶解）→ 调整时长0.3-0.5秒
  • 调色：选中片段 → 滤镜/调节 → 亮度+10~+20、对比度+10~+15

3.2 字幕制作

  • 点击"文字" → "自动字幕" → 选择音频 → 等待识别 → 修正错误字
  • 花字：点击"文字" → "花字" → 选择模板 → 修改文字

3.3 音频处理

  • 背景音乐：点击"音频" → "音乐" → 拖入时间轴 → 调整音量30%-50%
  • 降噪：选中音频 → "降噪" → 选择级别

3.4 竖屏横屏切换

格式平台导出设置
竖屏9:16抖音、小红书1080×1920
横屏16:9B站、西瓜视频1920×1080

四、平台运营技巧

4.1 小红书运营

  • 发布时间：早间7:00-9:00、午间12:00-14:00、晚间18:00-22:00
  • 发布频率：新账号每天1篇，连续7天；稳定后每天1-2篇
  • 标签策略：3-5个相关话题 + 1个热门话题 + 1个细分垂类

4.2 抖音运营

  • 完播率优化：前3秒必须有钩子，时长15-60秒最佳，节奏紧凑
  • 互动引导：结尾提问、引导评论、设置悬念促转发

4.3 违规红线

  • 绝对化词：最好、第一、唯一
  • 医疗/金融敏感词
  • 虚假夸大宣传
  • 搬运原创内容
  • 刷量买赞行为
